# CrankGPT — Enclosure CAD Files

CAD files for the CrankGPT enclosure.

## Files

| File | Description |
|------|-------------|
| `crankgpt_enclosure_top.step` | Top half of the enclosure |
| `crankgpt_enclosure_bottom.step` | Bottom half of the enclosure |

Files are provided in **STEP** (`.step`) format, which can be opened and
edited in most CAD tools (FreeCAD, Fusion 360, Onshape, SolidWorks, etc.)
and imported into slicers for 3D printing.
